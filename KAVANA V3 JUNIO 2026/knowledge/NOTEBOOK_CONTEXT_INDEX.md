# Índice de documentos NotebookLM

Este archivo debe mantenerse actualizado cada vez que se incorpore un documento generado con NotebookLM.

## Plantilla de entrada

```md
## Nombre del documento

- Ruta: `notebook/nombre_documento.md`
- Tema: producción / inventario / IA / costes / UX / estrategia / otro
- Utilidad: qué decisiones o reglas aporta
- Acciones: qué documentos de V3 debe actualizar
```
