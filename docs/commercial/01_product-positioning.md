# Kavana Manufacturing - Posicionamiento de Producto

## Estado del documento

- **Estado:** Documento comercial actualizado con dual theme, guías de usuario, y OEE como módulo opcional.
- **Última actualización:** 2026-07-04.

## Posicionamiento

Kavana Manufacturing se posiciona como un **MES SaaS táctil, offline-first y multi-tenant para plantas industriales que necesitan simplicidad operativa sin renunciar a seguridad y trazabilidad**.

## Público objetivo

- Fabricantes discretos.
- Talleres industriales.
- Plantas con operarios usando tablets o pantallas táctiles.
- Empresas que necesitan digitalizar producción sin implantar un ERP pesado.
- Organizaciones multiplanta que requieren segregación clara de datos.
- **Supervisores veteranos** que prefieren estilo ERP clásico.
- **Operarios jóvenes** que prefieren estilo moderno.

## Necesidades del cliente

- Registrar producción en tiempo real.
- Reducir errores manuales.
- Visualizar estado de órdenes.
- Medir tiempos y paradas.
- Operar aunque falle la red.
- Adaptar campos a cada sector.
- Mantener datos separados entre clientes o plantas.
- **Respetar preferencias visuales de usuarios diversos.**

## Mensaje comercial principal

**Kavana Manufacturing convierte la captura de producción industrial en una experiencia táctil, rápida y fiable, diseñada para operar incluso cuando la red de planta falla — con un sistema de temas que respeta la diversidad de usuarios industriales.**

## Pilares de venta

1. **Simplicidad operativa**
   - Flujos de 1 o 2 clics.
   - Interfaz de visión de túnel.
   - Botones grandes para operarios.

2. **Resiliencia industrial**
   - Offline-first.
   - Cola local de eventos.
   - Sincronización automática.

3. **Seguridad SaaS**
   - Multi-tenant.
   - Aislamiento de datos.
   - RLS y control por tenant.

4. **Dual Theme (Diferencial único)**
   - Clásico ERP: tablas, fondos claros — para usuarios tradicionales.
   - Moderno Kavana: tarjetas, fondos oscuros — para usuarios modernos.
   - Toggle flotante con persistencia en localStorage.

4. **Escalabilidad modular**
   - Core MES.
   - OEE.
   - Calidad.
   - Costes.
   - Feature flags por cliente.

5. **Adaptación cross-sector**
   - Campos personalizados.
   - Configuración por tenant.
   - Sin reescribir el núcleo.
