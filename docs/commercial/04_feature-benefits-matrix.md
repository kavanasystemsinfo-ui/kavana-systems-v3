# Kavana V3 - Matriz Funcionalidad / Beneficio

## Estado del documento

- **Estado:** Documento comercial actualizado con módulos premium integrados en UI.
- **Última actualización:** 2026-07-04.

| Funcionalidad | Beneficio para planta | Beneficio para negocio |
|---|---|---|
| Órdenes de producción | Visibilidad clara del trabajo en curso | Trazabilidad completa de lotes |
| Puestos de trabajo | Asignación precisa de recursos | Control de capacidad |
| Logs de tiempos | Registro de arranques, pausas y paradas | Base para OEE y costes |
| HMI táctil | Menos errores del operario | Mayor adopción del sistema |
| Offline-first | Continuidad ante fallos de red | Menos interrupciones productivas |
| Cola FIFO | Sincronización ordenada | Datos fiables y auditables |
| Multi-tenant | Datos separados por cliente | Modelo SaaS escalable |
| Feature flags | Módulos activables por cliente | Monetización modular |
| Custom fields | Adaptación por sector | Menor dependencia de desarrollos a medida |
| RLS | Seguridad a nivel de fila | Reducción de riesgo legal y operativo |
| **Dual Theme** | **Respetar preferencias de usuarios** | **Mayor adopción, menor resistencia al cambio** |
| **OEE automático** | **Identificar cuellos de botella en tiempo real** | **Mejorar OEE global de la planta** |
| **Quality checks** | **Registro consistente de defectos** | **Reducción de scrap y retrabajo** |
| **Cost tracking** | **Visibilidad de costos por categoría** | **Control de márgenes por producto** |
| **Cross-tenant isolation** | **Datos protegidos entre clientes** | **Cumplimiento normativo y legal** |

## Prioridad comercial

Las funcionalidades con mayor valor de venta inicial son:

1. HMI táctil simple.
2. Offline-first.
3. Multi-tenant seguro.
4. Trazabilidad de producción.
5. Modularidad premium.
6. **Dual Theme (Diferencial competitivo).**
7. **OEE automático (Módulo premium).**
8. **Quality + Cost (Módulos premium).**
