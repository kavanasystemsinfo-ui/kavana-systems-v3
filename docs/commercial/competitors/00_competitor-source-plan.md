# Competitor Source Plan

## Objetivo

Preparar fuentes limpias para NotebookLM sobre competidores de MES/manufactura y plataformas ERP/producción.

## Criterio mínimo

Cada fuente debe incluir:

- URL original.
- Fecha de captura.
- Versión o fecha del manual si está disponible.
- Cita breve.
- Dolor del usuario.
- Implicación para Kavana Manufacturing.

## Prohibido

- No inventar características.
- No copiar contenido protegido extenso.
- No incluir datos personales de reseñadores.
- No sincronizar borradores sucios con NotebookLM.
